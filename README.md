## ğŸ–¥ Introduction ğŸ–¥
This project is a 3D visualization of the mental disorders of Disney princesses, created using Three.js. The goal is to provide an interactive experience that allows users to explore the different mental disorders associated with each princess. You can navigate through the museum, and learn about their respective disorders. I also added a Karmine Corp touch to the project, as I am a big fan of the team. Here is the solution for the easter egg in case you need it: _At the end of the visit, you can clic 3 times on the "next" button to unlock the easter egg. You will be transported via a camera animation to the path of the basement. Clic again on the next button twice to see the easter egg._
